# Habit Flow

**Habit Flow** is a simple and interactive web app designed to help users track, manage, and visualize their daily habits. Build good habits consistently and monitor your progress over time with an easy-to-use interface.  

---

## Features
- Add, edit, and delete habits
- Mark habits as complete
- Visual progress tracking
- Color-coded habits for easy organization
- Responsive layout for desktop and mobile

---

## Tech Stack
- **HTML5** for structure
- **CSS3** for styling
- **JavaScript** for interactivity

---

## Usage
1. Open `index.html` in your browser or access the live GitHub Pages link.  
2. Start adding your daily habits.  
3. Mark habits as complete and watch your progress grow!
